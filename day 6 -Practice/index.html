Q 5 : Create an example to explain the difference between margin and padding

Margin

margin between the browsers screen and the element border

  • Margin Top
  • Margin Right
  • Margin Bottom
  • Margin Left

Padding

The space between the Element border and the Content

  • Padding Top
  • Padding Right
  • padding Bottom
  • padding Left

short hand property
margin:10px 10px
padding:10px 3px 4px
