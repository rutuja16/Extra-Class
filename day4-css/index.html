CSS Display Properties

Display-
its a just property in css which will tell us how to Display our element on the screen

Block-
it will try to occupy the complete width of our screen and it will always comes in new line

inline
it will try to come on same line and it will occupy how much width content is required.in inline the height and width are not to be considered

inline-block
it is an combination of both block and inline .it will come on same line only

None
which will completely deleted from our screen

visibility:hidden
element will be hidden from our screen but it is going to occupy the space

flex
it is an one diamentional where we are going to have either row or either column .we can adjust our element we can change the direction of our element by using the flex-direction

justify content
content align our items in main axis (x direction -row).

align-item
align items align our item in cross direction(y direction column
